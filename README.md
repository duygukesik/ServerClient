# Client-Server Chat Application (C)

This repository contains a simple client-server chat application written in C.  
The project is inspired by basic messaging systems and demonstrates fundamental
client-server communication logic.

## Features

- TCP-based client-server communication
- Message exchange between client and server
- Server-side logging
- Command-line interface

## Technologies

- C Programming Language
- Socket Programming (TCP)
- GCC or compatible C compiler
